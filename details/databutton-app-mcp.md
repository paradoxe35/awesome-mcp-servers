# Databutton App Mcp

A proxy MCP server that enables LLM clients to interact with Databutton app API endpoints via the MCP protocol over websockets.

## Features
- Acts as a local proxy MCP server for Databutton app APIs.
- Allows LLM clients (e.g., Claude Desktop, Cursor) to use Databutton app APIs as tools via MCP protocol over websockets.
- Secure connection to the Databutton app using API keys.
- Simple setup: requires downloading an API key and configuring the LLM client accordingly.
- Compatible with any MCP-compatible client.
