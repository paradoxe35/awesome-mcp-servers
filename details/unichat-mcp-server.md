# unichat-mcp-server

**Category:** AI Integration MCP Servers  
**Tags:** mcp, llm-integration, ai-integration, multi-platform

## Description
unichat-mcp-server is an MCP (Multi-Provider Chat Protocol) server that enables sending requests to multiple AI providers, including OpenAI, MistralAI, Anthropic, xAI, Google AI, DeepSeek, Alibaba, and Inception. It can be used via tools or predefined prompts, and requires you to provide the relevant vendor API key for each service.

## Features
- Send requests to a variety of AI providers (OpenAI, MistralAI, Anthropic, xAI, Google AI, DeepSeek, Alibaba, Inception)
- Supports MCP protocol for multi-provider integration
- Usable via tools or predefined prompts
- Requires vendor API key for each provider
- Available in Python (also in TypeScript)
- Can be integrated with Claude Desktop (configuration instructions provided for MacOS and Windows)
- Supported model list provided for selection
- Can be installed via Smithery (for Claude Desktop integration)
- Debugging support via MCP Inspector (launchable via npm, browser-accessible interface)
- Development and publishing instructions included (builds source and wheel distributions)
