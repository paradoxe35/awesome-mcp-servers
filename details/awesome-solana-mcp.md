# Awesome Solana MCP

A Model Context Protocol (MCP) server enabling standardized interaction with the Solana blockchain, designed for integration with Claude AI assistants. Built on the Solana Agent Kit, this server provides a unified interface for various blockchain operations.

## Features
- **Validator Information**: Retrieve detailed data on Solana validators, including stake amounts, commission rates, and performance metrics.
- **Priority Fee Estimation**: Get optimal transaction fee estimates (low/medium/high tiers) based on current network conditions with expected confirmation times.
- **Transaction History**: Fetch transaction histories for any Solana wallet or token account, with support for transaction type filtering and pagination.
- **Security Contact Extraction**: Extract security contact information from Solana programs to assist with vulnerability reporting.
- **Asset Management**: Perform asset-related operations such as retrieving asset details and minting NFTs.
- **Token Operations**: Deploy new tokens, transfer tokens, and trade tokens on the Solana network.
- **Network Information**: Access network metrics such as transactions per second (TPS) and wallet balances.
- **Utility Functions**: Request test funds (airdrop), and register domains on Solana.
