# multi-ai-advisor-mcp

**Category:** AI Integration MCP Servers  
**Tags:** ollama, llm-integration, multi-model, open-source

**Description:**

multi-ai-advisor-mcp is a Model Context Protocol (MCP) server that queries multiple Ollama models in parallel, each with distinct system prompts focused on empathy, logic, and creativity. It combines the responses from these models to provide diverse AI perspectives via a single MCP server endpoint.

**Source:** [multi-ai-advisor-mcp on MagicSlides](https://www.magicslides.app/mcps/multi-model-advisor)

## Features
- Queries multiple Ollama models in parallel
- Uses distinct system prompts for each model (e.g., empathy, logic, creativity)
- Combines responses from different models into a single output
- Exposes a unified MCP server endpoint for easy integration
- Can be installed via Smithery or manually
- Configuration options available
- Supports connection to Claude for Desktop
- Allows listing of available models
- Includes basic usage instructions
- Troubleshooting guidance for common issues (model not found, RAM issues, etc.)
- Open-source
